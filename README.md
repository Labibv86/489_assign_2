Project Overview

This Android application is designed to demonstrate core Android platform development concepts, specifically handling Local Media Architecture (Video & Audio) and Global/Local Event Communication Broadcasting. The app uses a modern single-activity navigation architecture with a side NavigationView Drawer menu to switch seamlessly between isolated functional modules.

All media components are fully optimized for local, offline execution to bypass network-related emulator bottlenecks. The communication framework implements a multi-stage workflow conforming to modern Android 14+ sandboxing and security guidelines.



Project Architecture & Major File Directory

The application components are decoupled cleanly into structural layout blueprints XML and programmatic controllers Java.
